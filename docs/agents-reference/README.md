# Agents Reference

These are **reference/learning materials** for studying advanced agent architecture patterns. They are **NOT** active Claude agents and are not executed by DailyLoadout. They originate from Tier 3 CleanMarket agent definitions and have been preserved here for theoretical study and design inspiration.

## Categories

### Consensus (`consensus/`)

Distributed consensus protocol agents covering fault tolerance, state replication, and secure coordination across distributed nodes.

| File | Description |
|------|-------------|
| `crdt-synchronizer.md` | Conflict-free Replicated Data Types for eventually consistent state synchronization |
| `raft-manager.md` | Raft consensus algorithm with leader election and log replication |
| `byzantine-coordinator.md` | Byzantine fault-tolerant consensus with malicious actor detection |
| `gossip-coordinator.md` | Gossip-based consensus for scalable eventually consistent systems |
| `quorum-manager.md` | Dynamic quorum adjustment and intelligent membership management |
| `security-manager.md` | Cryptographic security mechanisms for distributed consensus protocols |
| `performance-benchmarker.md` | Performance benchmarking and optimization for consensus protocols |
